# GET Academy – Kodeeksempler og Components

<p>Dette repoet inneholder kodeeksempler brukt i videoer fra <b>GET Academy</b> på sosiale medier.
Hver mappe i repoet representerer én video eller én komponent, og viser konkrete eksempler på <b>HTML</b>, <b>CSS</b> og <b>JavaScript</b>.</p>

<p>Målet er at du enkelt skal kunne:</p>
<ul>
  <li>👀 Se hvordan ting fungerer i praksis</li>
  <li>🧪 Eksperimentere selv</li>
  <li>🚀 Bruke koden i egne prosjekter</li>
</ul>

<h2>🛠 Hvordan bruke koden selv</h2>
<h3>1️⃣ Last ned repoet</h3>
<b>Alternativ A</b> – Clone med Git

```bash
git clone https://github.com/RebeckaKinn/tiktok_video_components.git
```

<b>Alternativ B</b> – Last ned ZIP
<ul>
  <li>Trykk på Code → Download ZIP</li>
  <li>Pakk ut på PCen din</li>
</ul>

<h3>2️⃣ Åpne prosjektet</h3>
<ul>
  <li>Åpne mappen i VS Code eller annen editor</li>
  <li>Åpne index.html i nettleser</li>
</ul>

<h3>Tips i VS Code:</h3>
<ul>
  <li>Installer extension Live Server</li>
  <li>Høyreklikk → Open with Live Server</li>
</ul>
